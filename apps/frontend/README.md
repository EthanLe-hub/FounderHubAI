# FounderHQ Frontend

This is the Next.js 13+ frontend for the FounderHQ AI-powered co-pilot platform.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. The app will be available at [http://localhost:3000](http://localhost:3000)

## Tech Stack
- Next.js 13+
- TypeScript
- Tailwind CSS
- Shadcn/ui
